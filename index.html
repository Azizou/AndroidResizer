Androidresizer

A java Desktop application to resize pictures for android developement ie ldpi, mdpi, hdpi, xhdpi, xxhdpi

Download Download View Androidresizer on GitHub

Making an android app and you need to resize images to place into those drawable folders? ANDRE(Android Resizer) is a simple desktop application to do just that, reduce the time you take to resize the images to the respective resolutions. Requirements: Ensure to have Java 8 jre installed from here

Instructions: Download the ANDRE.jar file from the dist file. Run the jar file. Select the images you want to resize and your destination folder containing your drawable folders(the \res folder of app) Enjoy and keep resizing..ANDRE at your service!
